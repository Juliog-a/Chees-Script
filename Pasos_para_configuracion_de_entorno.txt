Abrir el proyecto en VSCode

Abrir terminal con Cntrol+Ã±

Dividir el terminal en dos, uno para el backend y otro para el frontend:
	cd backend

	cd frontend

Configuramos ambos:
	backend:
		Creamos el entorno virtual: python -m venv venv

		Activamos el entorno virtual: .\venv\Scripts\Activate

		Instalamos los requerimientos: pip install -r requirements.txt


	frontend: npm install


Iniciamos cada uno:
	backend: python manage.py runserver
	
	frontend: npm run dev



Para subir a GitHub la primera vez:
	git init
	git add .
	git commit -m "mensaje"
	git push --set-upstream https://github.com/Juliog-a/Chees-Script.git master
